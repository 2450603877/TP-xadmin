# Typecho-x-admin
此模板基于 X-admin 开发
创意来源 [小凡梦](http://www.xiaofm.cn/index.php/archives/48/)
本代码基于 MIT 协议开源

-------
#### 模板特点
1. 管理页面完美内嵌
2. 支持 Typecho 多级权限
3. 轻量级后台
4. 开源免费
